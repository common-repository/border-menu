WordPress menu plugin for creating animated border menu with a sticky hamburger button

== Description ==
WordPress sticky menu plugin 'Border Menu' with hamburger button for creating animated border menu. The Border Menu is the easy-to-use Wordpress menu plugin for setting up the unfolding menu through the "hamburger menu icon". The tool allows you to create a convenient and compact navigation bar on the site. It is used only when necessary and does not distract the attention of users

= Demo =

[DEMO](https://wow-estore.com/item-preview/border-menu/)

= Main features =
The Border Menu plugin will give recognition to the design of your site. The extension allows to:

* create 1 menu;
* select 4 items for the main button;
* insert any links;
* display modal windows through the insertion of shortcodes; 
* use more than 600+ Font awesome icons;
* customize the color of the menu buttons;
* and more...

= Can be used for =
* site navigation;
* additional menu;
* user panel;
* block of contacts and much more.

The Border Menu plugin is the convenient and simple tool for creating an unfolding menu on the border of the page. Navigation through the "hamburger icon" focuses the attention of users on the content of the resource. Free expansion will allow you to try a new approach in developing the structure of the site. Increase the conversion by reducing the number of distractions.

== Installation ==
* Installation option 1: Find and install this plugin in the `Plugins` -> `Add new` section of your `wp-admin`
* Installation option 2: Download the zip file, then upload the plugin via the wp-admin in the `Plugins` -> `Add new` section. Or unzip the archive and upload the folder to the plugins directory `/wp-content/plugins/` via ftp
* Press `Activate` when you have installed the plugin via dashboard or press `Activate` in the in the `Plugins` list 
* Go to `Border Menu` section that will appear in your main menu on the left
* Click `Add new` to create your first menu
* Setup your menu
* Click Save
